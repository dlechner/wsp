Common Playstation 3 Light and Error Codes

  • Yellow Light: This is a severe hardware failure. Complete mainboard and system diagnostics are required to determine the root cause
  • Red Blinking Light: This is commonly caused due to the system over heating. Verify sufficient ventilation of the console. If the problem persists then a diagnostics will need to be performed.
  • Lights ok, will not read discs: The blue-ray laser aperature is malfunctioning and will need to be replaced.
  • No Lights: The power supply cable is loose or malfunctioning. Verify proper connection, if still no power/lights then try a new power cable. If the problem still persists then a diagnostics will be needed.
  • 80010514-Issue Reading Discs: Damaged or dirty disc, clean disc or try a different disc. If problem persists with clean discs then a disc drive diagnostics will be needed.
  • 80710B23-Can't Connect to Playstation Network: Verify online to see if PSN is down for maintenance. Verify account settings.
  • 8002A524-System memory low: Sometimes data can become corrupted stored in the RAM. Simply reboot the system to clear the corrupted data.
  • 8071-Common Error Code: Can be from faulty internet connections to hardware problems. A full diagnostics will be needed.

If problems persist with any of the above codes a full diagnostics is needed to confirm and repair accordingly.
